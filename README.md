# frontend-project-44
## Brain-games is a simple set of console mini-games.
### Requirements:

  Node-JS v22.8.0 or higher.

### How to install games:
#### Open console and write:
  To clone repository:
  ```sh
git clone https://github.com/Vova0211/frontend-project-44
```
  To install project:
  ```
make install
```
  ```
make publish
```
  ```
npm link
```

### Now you can play Brain games!
